Reference

Titles

First line of each section MUST be a title
Then in a new line insert the content of that section
Dont leave a blank line, and start the document always with a title
eg: {THIS IS A TITLE}

Links

Can be anywhere in a line
Start with [ and then add the text of the link
on the same line add a space and path:http.....
and a ] at the end of the path
eg: [this is a link path:https://goo.gl/gdljno]

Lists

First item of a list MUST start with #
Consecuent items MUST be in a single line
You can nest list by starting another #
Close the list with /# in a single line

Tables

Start with a single line <t>
Mark the table header with a th in the beggining
Separate column elements with a TAB
Close the table with </t>
REMEMBER TO CLOSE THE TABLE

Buttons

MUST be alone in a single line
Start with ¬ and then add the text of the button
on the same line add a space and path:http.....
eg: ¬button path:https://goo.gl/gdljno

Images

MUST be alone in a single line
@ for full width img
@l left side img position
@r right side img position
You can add a footer to the image by using ()
eg: @https://goo.gl/gdljno (this is the image footer)
